 \doxysection*{Description}

The map folder has some pre-\/configured maps for you to test the game environment. ~\newline
 You\textquotesingle{}re able to create a map of your own and the program will tell you if the map is valid or not, pointing the out the problem if it isn\textquotesingle{}t. ~\newline




{\bfseries{Click here for map creation tutorial.}}\begin{adjustwidth}{1em}{0em}


Map Rules\+: ~\newline
 It has to be a {\bfseries{rectangular closed map}} (walls around) and there needs to be a valid way to collect the health packs and to the exit. ~\newline
 The map file extension has to be \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\textquotesingle{}.ber\textquotesingle{}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}. ~\newline
 The map is composed by 5 elements\+: ~\newline
 
\begin{DoxyCode}{0}
\DoxyCodeLine{-\/\ '1'\ for\ walls;}
\DoxyCodeLine{-\/\ '0'\ for\ available\ floor;}
\DoxyCodeLine{-\/\ 'P'\ for\ Player\ starting\ position;}
\DoxyCodeLine{-\/\ 'E'\ for\ exit;}
\DoxyCodeLine{-\/\ 'C'\ for\ collectible;}

\end{DoxyCode}

To run the program, you need to select a map from the existing ones or the one that you\textquotesingle{}ve created. For that, you need to put its filename, extension and directory. ~\newline


For {\itshape mandatory}\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$\ ./so\_long\ maps/map.ber}

\end{DoxyCode}
